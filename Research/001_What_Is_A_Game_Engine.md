# Research 1:

### What is a game engine?
*Game engine is a frame work used to create and publish games, Also include specilized software libraries and package.*

### Why do fames use engine?
*Game engine provide prebuild reuasable systems like rendering, Physics and AI which saves many years and also avoid developers to write core infrastructure from scratch.*

### Hoe do Unity and Unreal work internally?
*Unity relies on a lightweight component based architecture driven by c#.*
*Unreal engine use a heavier framework written in c++.*

### What modules does game engine have?

1. Rendering Module
2. Physics Engine
3. Audio System
4. Input Handling & Core system
5. Asset Management
6. Networking Module
7. Animation Engine

### Why build a game engine from scratch?
*In my case to learn deep about opengl and game engine architecture and also to get full control over a engine.*
